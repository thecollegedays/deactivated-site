Internet Terminologies

TCP
Transmission Control Protocol is a method used along with the Internet Protocol to send data in the form of message units, called packets between computers and the internet
IP
Internet protocol is the method by which data is sent from one computer to another on the internet. Each computer is uniquely identfied by an IP Address
